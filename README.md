# Vagrantfiles

## Vagrantコマンド

```
# 仮想マシンの起動
vagrant up

# 仮想マシンの停止
vagrant halt

# 仮想マシンの再起動(停止 & 起動)
vagrant reload

# 仮想マシンの状態を一覧表示
vagrant global-status

# 仮想マシンにssh接続
vagrant ssh
```
